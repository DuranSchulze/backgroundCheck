This is a minimal [Next.js](https://nextjs.org) project set up with TypeScript, Tailwind CSS, and the App Router.

## Getting Started

Install dependencies if needed, then run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Start editing by updating `app/page.tsx` and `app/layout.tsx`.

## Included

- Next.js App Router
- TypeScript
- Tailwind CSS
- ESLint

## Useful Commands

```bash
npm run dev
npm run lint
npm run build
npm run start
```

For more details, visit the [Next.js documentation](https://nextjs.org/docs) and the [Tailwind CSS documentation](https://tailwindcss.com/docs).
